🛒 Shopping Cart System (Activity)📝 Project DescriptionThis is a console-based Shopping Cart System developed in C#. The application allows users to interact with a store menu, select items, and manage a virtual cart while simulating real-time inventory management. It is designed to demonstrate concepts of Object-Oriented Programming (OOP) through the use of classes, objects, and arrays.✨ Key FeaturesMenu Interaction: Displays a list of products with their respective IDs, names, prices, and remaining stock.Inventory Validation: Prevents users from purchasing more than the available stock and handles out-of-stock items (RemainingStock = 0).Smart Cart Management: Instead of adding duplicate rows, the system updates the quantity and subtotal if the same product is selected again.Robust Input Handling: Uses int.TryParse() to prevent system crashes from non-numeric inputs.Automated Discount: Applies a 10% discount if the Grand Total reaches 5,000 or more.Persistent Tracking: Displays the updated stock of all items after the checkout process is completed.🛠️ Technical DetailsClass Structure: Product The system is built around a Product class with the following components:Fields: Id, Name, Price, and RemainingStock.Methods:DisplayProduct(): Shows item details.GetItemTotal(int quantity): Computes cost per item.HasEnoughStock(int quantity): Validates availability.DeductStock(int quantity): Updates stock counts after a valid purchase.📊
